# Equi'Roul Club

Site web de l'école d'équitation Equi'Roul (Monnaz, Suisse).

## Structure

- `index.html` — Application (HTML + CSS + JS)
- `vercel.json` — Configuration Vercel
- `videos/` — Vidéos des poneys
- `images/` — Miniatures

## Déploiement

Déployé automatiquement sur Vercel à chaque push.
